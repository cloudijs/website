# Cloudijs Website

This repository contains the source code for [www.cloudijs.nl](https://www.cloudijs.nl).
Currently this is only a placeholder.

## Deployment

Static website files are automatically deployed to Bunny Storage and Bunny CDN
when they are updated on the **main** branch. Props go to the [bunsy](https://codeberg.org/sbstjn/bunsy/src/branch/main)
project for making uploading the files in the GitHub Actions pipeline easy.
